## Getting Started

Clone repo to local machine using git:

```
$ git init
```
```
$ git clone https://github.com/whitebrandon/techdegree-project-8
```

Install dependencies via npm:

```
$ npm install
```

Run project via node:

```
$ node app.js
```

or npm:

```
$ npm start
```
Navigate to `localhost:3000` in web browser
